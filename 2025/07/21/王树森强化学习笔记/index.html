1、RL术语

X是随机变量，x是观测值

state s

Action a，a是agent做的，以马里奥为例，left right up

policy π\piπ π:(s,a)=p(A=a∣S=s)\pi:(s,a)=p(A=a|S=s)π:(s,a)=p(A=a∣S=s) 即给定状态s，做出动作a的概率

reward 以马里奥为例，捡到金币+1，赢了+1w，输了扣1w

state transition 状态转移

强化学习中两个随机性来源：动作的随机性，状态转移的随机。随机产生a，随机产生s

trajectory：每一步是s,a,rs,a,rs,a,r 直到游戏结束有s1,a1,r1,s2,a2,r2,...sT,aT,rTs_1,a_1,r_1,s_2,a_2,r_2, ... s_T,a_T,r_Ts1​,a1​,r1​,s2​,a2​,r2​,...sT​,aT​,rT​

Return: cumulative future reward RtR_tRt​ 是当前的回报，未来的回报具有不确定性，所以加了一个超参数

Ut=Rt+γRt+1+γ2Rt+2+...U_t = R_t+\gamma R_{t+1} + \gamma^2 R_{t+2} + ... Ut​=Rt​+γRt+1​+γ2Rt+2​+...

Acition-Value Function for policy π\piπ Q(s,a)Q(s,a)Q(s,a) ，对UtU_tUt​的期望，因为它与当前t时刻与未来每个时刻的A和S都有关，求期望得到动作价值函数的定义

Qπ(s,a)=E[Ut∣St=st,At=at]Q_\pi(s,a) = \mathbb{E}[U_t| S_t =s_t,A_t=a_t]Qπ​(s,a)=E[Ut​∣St​=st​,At​=at​]

Optimal action-value function Q最大的时候

Q∗(s,a)=maxπQπ(st,at)Q^*(s,a) =\mathop{max}\limits_{\pi} Q_\pi (s_t,a_t)Q∗(s,a)=πmax​Qπ​(st​,at​)

State-Value function

Vπ(S)=EA[Qπ(s,A)]V_\pi(S) = \mathbb{E_A}[Q_\pi(s,A)]Vπ​(S)=EA​[Qπ​(s,A)]

总结就是，动作价值函数是状态sts_tst​下，采取动作ata_tat​，能够得到多少回报，当然这个值越大越好。而状态价值函数是只固定了状态sts_tst​，动作A是随机，因此状态价值函数的结果是，当前状态下，能够得到的平均回报

强化学习任务，一是学习策略函数，二是学习Q函数。学习两者之一即可。

value-based learning： DQN，任务是最大化Q 用到temporal different TD算法

Policy-based learning ：任务是最大化π\piπ 用到policy gradient，梯度上升

Actor-critic method: policy network + value network

2、Value-Based RL

用一个神经网络去近似Q∗(s,a)Q^{*}(s,a)Q∗(s,a)

如果我们知道了Q∗(s,a)Q^*(s,a)Q∗(s,a)那么很明显最优动作就是

但是我们现在没有Q∗(s,a)Q^*(s,a)Q∗(s,a) 所以只能用一个神经网络去近似

但是怎么训练？用TD训练，目的是让TD error越小越好

类比driving time例子，TD公式一般类似于这样,左边是T时刻的估计，右边是TD target，包含一个真实值和T+1时刻的估计

TNYC−>ATL≈TNYC−>DC+TDC−>ATLT_{NYC->ATL} \approx T_{NYC->DC} + T_{DC->ATL}TNYC−>ATL​≈TNYC−>DC​+TDC−>ATL​

而在强化学习中

Q(st,at;w)≈rt+γ⋅Q(st+1,at+1;w)Q(s_t,a_t;w) \approx r_t + \gamma \cdot Q(s_{t+1},a_{t+1};w) Q(st​,at​;w)≈rt​+γ⋅Q(st+1​,at+1​;w)

yt=rt+γ⋅Q(st+1,at+1;wt)y_t = r_t + \gamma \cdot Q(s_{t+1},a_{t+1};w_t)yt​=rt​+γ⋅Q(st+1​,at+1​;wt​)

y_t = r_t + \gamma \cdot \max_\limits{a} Q(s_{t+1},a;w_t)

注意到这里，为什么要求一个max？我的理解是，因为我们的目标就是训练一个最优的Q？

3、Policy-Based RL

策略函数π(a∣s)\pi(a|s)π(a∣s)是一个概率分布函数 给定状态s，做出动作a

如果状态有限，直接用一个表记录就行，超级马里奥需要用函数去近似

回顾一下状态价值函数

Vπ(st)=EA[Qπ(st,A)]=∑aπ(a∣st)⋅Qπ(st,a)V_\pi(s_t) = \mathbb{E_A}[Q_\pi(s_t,A)]=\sum_a\pi(a|s_t) \cdot Q_\pi(s_t,a)Vπ​(st​)=EA​[Qπ​(st​,A)]=∑a​π(a∣st​)⋅Qπ​(st​,a)

通过神经网络去近似策略函数π(a,st;θ)\pi(a,s_t;\theta)π(a,st​;θ)

V(st;θ)=∑aπ(a∣st;θ)⋅Qπ(st,a)V(s_t;\theta) =\sum_a\pi(a|s_t;\theta) \cdot Q_\pi(s_t,a)V(st​;θ)=∑a​π(a∣st​;θ)⋅Qπ​(st​,a)

Policy-based learning：Learn θ\thetaθ that maximizes J(θ)=ES[V(S;θ)]J(\theta) = \mathbb{E_S}[V(S;\theta)]J(θ)=ES​[V(S;θ)]

那么算policy gradient的时候，需要用到梯度上升，不严谨的推导可以得到两个公式，分别对应离散和连续

∂V(s;θ)∂θ=∑a∂π(a;s,θ)∂θ⋅Qπ(s,a)\frac{\partial V(s;\theta)}{\partial \theta} = \sum_a \frac{\partial \pi(a;s,\theta)}{\partial \theta} \cdot Q_\pi(s,a)∂θ∂V(s;θ)​=∑a​∂θ∂π(a;s,θ)​⋅Qπ​(s,a)

∂V(s;θ)∂θ=EA∼π(⋅∣s;θ)[∂log⁡π(A∣s,θ)∂θ⋅Qπ(s,A)]\frac{\partial V(s;\theta)}{\partial \theta} = \mathbb{E_{A\sim\pi(\cdot|s;\theta)}}[\frac{\partial \log\pi(A|s,\theta)}{\partial \theta} \cdot Q_\pi(s,A)]∂θ∂V(s;θ)​=EA∼π(⋅∣s;θ)​[∂θ∂logπ(A∣s,θ)​⋅Qπ​(s,A)]

这里策略函数很复杂，积分求不出来，只能用MC去近似期望，MC就是抽一个或很多个样本，去近似期望
